圣诞节快到了！准备好了吗？是时候来一场与新语言的邂逅了！

你已经解决了Eric Wastl上神奇的谜题集锦 “代码挑战” 了吗？得益于Kotlin具有函数式语言的特性，这些任务可以找到非常简洁而不失可读性的解决方案。
尽力用kotlin去解决他们吧！

你可以在IntelliJ IDEA社区版上完成你的解决方案，它是免费的，也原生支持Kotlin。
